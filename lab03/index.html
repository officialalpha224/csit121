My First Webpage

At first, the process seemed daunting, with so many tags and structures to remember, but breaking it down step by step made it manageable. Learning how to properly structure an HTML document helped me understand how webpages are built and displayed.

What I Learned

  1. I learned how to make headers
  2. I learned how to make Ordered and Unordered lists
  3. I also learned how to make external links

Learning about Ordered lists, Unordered lists, and external links was another exciting step. I learned a lot from:

  • MDN HTML Guide
  • W3Schools HTML Tutorial

Additional Lessons Learned

I've learned what CSS is and what it stands for: Cascading Style Sheets. It controls the style (colors, fonts, spacing) and layout of web pages. I've also learned how to use Id selectors and Class selectors to apply specific styles to elements on a webpage. This allows for more precise and efficient styling, making it easier to create visually appealing and well-structured web pages.

This website has helped me have a deeper understanding CSS and selectors.
